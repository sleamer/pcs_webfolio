WordPress

Geekhub.com is a website built using worpress as the CMS. Fantasy, Horror, and Sci-fi are filtered to specific content templates that showcase customability in the category templates. Plugins include 'Events Manager', 'Contact Form 7', and 'Responsive Ad Ons'. The parent theme is Parallax by CyberChimps.

Partner: Mario Torres

MODx

A new age massage clinic website built on the backbone of MODx. getResources and Wayfinder were added snippits, while most files (Chunks, Templates, and Template Variables) were set to static and saved under git's version controls. Database exports were saved as flat files and versioned as well.

Partner: Xander Taylor
